## Debug步骤建议
1. 前置条件: 确认频偏测量与系统约束条件。
   - 确认频偏测量方法与工况（温度、电压、老化后漂移）
   - 确认I210工作模式与链路速率（10/100/1000）、是否有PTP/时间同步要求（如有）
2. 复现: 在受控环境复现+100ppm条件下的功能与稳定性数据。
   - 在不同温度点测量实际频率与ppm，并记录链路稳定性
   - 执行长时间吞吐/误码/掉线统计测试
3. 定位: 定位超频偏可能影响的路径（PHY/PCS、时钟恢复、协议容限）。
   - 检查链路协商、FEC/PCS错误计数（如可获取）
   - 检查是否在特定对端设备/线缆条件下更易出现边界问题
4. 验证: 验证将晶体恢复到规格范围后的差异，评估风险。
   - 更换合规ppm晶体进行A/B对比（掉线率、误码、性能）
   - 在最差工况（高温/低温/电压边界）下复测
5. 收敛: 形成结论：是否可接受偏离与需满足的验证/声明。
   - 若仍采用+100ppm：明确为超规格使用并给出风险说明与必要验证覆盖
   - 记录客户现网400+设备测试的测试项、时长与失败标准以形成可追溯证据
